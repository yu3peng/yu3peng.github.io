Qt Creator

note

快捷键（for mac）

参考：http://doc.qt.io/qtcreator/creator-keyboard-shortcuts.html

C –> Ctrl

S –> Shift

M –> Alt

Cmd –> Command

功能快捷键
自动完成C-Space
显示/隐藏侧边栏Cmd-0
切换已打开的文件M-Tab
上/下一行C-p/C-n
前进/后退一个字符C-f/C-b
删除一个单词M-Del
构建Cmd-b
运行Cmd-r
调试Cmd-y
注释Cmd-/
换行Cmd-Return
跳到定义F2
切换头文件与源文件F4
前进/后退M-Cmd-Left/Right
打开定位器Cmd-k
